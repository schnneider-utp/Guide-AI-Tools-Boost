# Frontend Design Skill

La skill **Frontend Design** está diseñada para elevar drásticamente la calidad estética y creatividad en la construcción de interfaces de usuario web, alejándose del diseño predecible y estandarizado comúnmente generado por las inteligencias artificiales (conocido como "AI slop").

## 🌟 ¿Qué hace?
Guía a la IA para crear interfaces frontend llamativas, memorables y de nivel de producción. Impulsa decisiones audaces en cuanto a tipografía, paletas de color, composición espacial y uso del movimiento (animaciones). 

## ⚙️ ¿Cómo funciona?
Fuerza al agente a adoptar un "Design Thinking" estructurado antes de escribir una sola línea de código:
1. **Elección de Tono y Estética**: Obliga a decidir un tono específico (minimalismo brutal, caos maximalista, retro-futurista, lujo, etc.) y comprometerse con él en todo el diseño.
2. **Rechazo a los Estándares Aburridos**: Instruye explícitamente a evitar fuentes sobreutilizadas (como Arial, Inter, Roboto), esquemas de color clicheados (como degradados morados sobre blanco) o layouts repetitivos.
3. **Refinamiento de Detalles**: Promueve el uso de variables CSS, animaciones de micro-interacciones de alto impacto, asimetría controlada y la adición de texturas, ruidos (noise) y mallas de gradientes para dar atmósfera.

## 🚀 ¿En qué puede mejorar tu proyecto?
- **Resultados Visualmente Únicos**: Tus componentes de React, Vue o simplemente HTML/CSS dejarán de parecer "creados por IA" y empezarán a sentirse como sitios web premiados o de alto nivel de diseño.
- **Implementación Compleja Adecuada**: Si pides un diseño maximalista, la IA entregará código elaborado con efectos avanzados; si pides uno minimalista, se enfocará en la precisión absoluta del espaciado y la tipografía.
- **Código Listo para Producción**: No sacrifica la accesibilidad o el rendimiento en favor de la estética, sino que entrega código funcional (como el uso de librerías como *Motion*) apegado a la visión visual.

