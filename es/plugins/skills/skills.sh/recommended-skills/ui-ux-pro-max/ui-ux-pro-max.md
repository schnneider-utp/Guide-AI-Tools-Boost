# UI/UX Pro Max Skill

La skill **UI/UX Pro Max** actúa como un sistema de inteligencia de diseño integral para aplicaciones web y móviles, proporcionando un vasto catálogo de directrices de usabilidad, estilos y accesibilidad.

## 🌟 ¿Qué hace?
Proporciona a tu IA un acceso instantáneo a más de 50 estilos visuales, 97 paletas de colores, 57 combinaciones de fuentes y 99 pautas de UX a lo largo de 9 stacks tecnológicos distintos (como React, Tailwind, Next.js, Vue, SwiftUI, etc.). Sirve como una referencia exhaustiva para que la IA tome decisiones de diseño robustas y accesibles.

## ⚙️ ¿Cómo funciona?
A diferencia de otras skills que solo dictan "cómo actuar", esta skill funciona de forma modular con base en reglas jerárquicas:
1. **Generación de Sistemas de Diseño**: A través de búsquedas (o uso de reglas internas) en distintos dominios (producto, estilo, color, tipografía), la IA extrae e implementa la mejor combinación para las necesidades del usuario.
2. **Reglas Basadas en Prioridades**: Valida el trabajo contra un riguroso set de prioridades, donde la **Accesibilidad** y la **Interacción/Táctil** son críticas, seguido por el rendimiento, responsividad y otros detalles.
3. **Persistencia del Diseño**: Genera y obedece archivos `MASTER.md` y de páginas individuales dentro de una carpeta de sistema de diseño en el proyecto, para que la IA mantenga un seguimiento continuo y sepa exactamente qué reglas aplicar en todo momento.

## 🚀 ¿En qué puede mejorar tu proyecto?
- **Accesibilidad Inmediata**: Aplica directamente contrastes correctos, anillos de foco (focus rings), navegación por teclado y `aria-labels`, logrando productos inclusivos y estándar.
- **Interacciones Táctiles y UX**: Considera cosas tan específicas pero vitales como el tamaño mínimo de objetivos táctiles (44x44px), feedbacks visuales ante errores y cursores interactivos.
- **Rendimiento Visual**: Mejora la carga al considerar uso de WebP, estado de carga (skeletons) y limitación de animaciones cuando es necesario.
- **Versatilidad Multi-Stack**: Ya sea que trabajes en Web con Tailwind, o en móvil con Flutter o SwiftUI, la IA conocerá las mejores prácticas UX para ese entorno específico.
- **Chequeo Automático Pre-Entrega**: Promueve el uso de listas de verificación exhaustivas (Pre-Delivery Checklist) antes de dar por terminado cualquier fragmento de código UI.

## 🛠️ Instalación y Enlaces Oficiales

Para instalar esta skill en tu proyecto a través de `skills.sh`, utiliza el siguiente comando:

```bash
npx skills add https://github.com/nextlevelbuilder/ui-ux-pro-max-skill --skill ui-ux-pro-max
```
