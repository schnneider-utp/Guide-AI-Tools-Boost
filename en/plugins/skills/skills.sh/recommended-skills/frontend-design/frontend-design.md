# Frontend Design Skill

The **Frontend Design** skill is designed to drastically elevate the aesthetic quality and creativity in the construction of user interfaces, moving away from the predictable, standardized design commonly generated by AIs (known as "AI slop").

## 🌟 What does it do?
Guides the AI to create striking, memorable, and production-grade web user interfaces. Drives bold decisions regarding typography, color palettes, spatial composition, and the use of motion (animations).

## ⚙️ How does it work?
Forces the agent to adopt structured "Design Thinking" before writing a single line of code:
1. **Tone & Aesthetic Choice**: Forces to decide on a specific tone (brutal minimalism, maximalist chaos, retro-futuristic, luxury, etc.) and commit to it throughout the entire design.
2. **Rejection of Boring Standards**: Explicitly instructs to avoid overused fonts (like Arial, Inter, Roboto), clichéd color schemes (like purple gradients on white backgrounds), or repetitive layouts.
3. **Detail Refinement**: Promotes the use of CSS variables, high-impact micro-interaction animations, controlled asymmetry, and the addition of textures, noise, and gradient meshes to create atmosphere.

## 🚀 How can it improve your project?
- **Visually Unique Results**: Your React, Vue, or simply HTML/CSS components will stop looking "AI-created" and start feeling like award-winning or high-design websites.
- **Adequate Complex Implementation**: If you ask for a maximalist design, the AI will deliver elaborate code with advanced effects; if you ask for one minimalist, it will focus on absolute precision in spacing and typography.
- **Production-Ready Code**: It doesn't sacrifice accessibility or performance for aesthetics, but delivers functional code (like using libraries like *Motion*) attached to the visual vision.

## 🛠️ Installation & Official Links

To install this skill in your project through `skills.sh`, use the following command:

```bash
npx skills add https://github.com/anthropics/skills --skill frontend-design
```
